=================================
User interface content guidelines
=================================

For information about using Helix for user interface design and development,
see the following references:

- `Helix design guide <http://helix.rax.io/>`_
- `Helix developer guide <https://rackerlabs.github.io/helix-ui/guides/getting-started/>`_

Use these user interface (UI) text guidelines if you are a designer,
developer, or reviewer contributing content in Rackspace UIs.
These UI text guidelines ensure that a Rackspace UI is usable,
consistent, and concise.
